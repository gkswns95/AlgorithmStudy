### 나의 풀이

```python
import math

n = int(input())
for _ in range(n):
    ret = 0
    test_case = list(map(int, input().split()))
    for i in range(1, len(test_case)):
        a = test_case[i]
        for j in range(i+1, len(test_case)):
            b = test_case[j]
            
            ret += math.gcd(a, b)
    print(ret)
```
